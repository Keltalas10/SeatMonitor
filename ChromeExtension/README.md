# Seat Monitor - Chrome Extension

Расширение Chrome для автоматического мониторинга и занятия места за столом.

## Установка
1. Разархивируйте архив с extenstion в удобное место (после установки удалять файлы нельзя иначе extention перестанет работать)
2. Откройте Chrome и перейдите по адресу: chrome://extensions/
3. Включите "Режим разработчика" (Developer mode) в правом верхнем углу
4. Выберите "Загрузить распакованное расширение" и укажите путь к
6. Расширение установлено!

## Использование

1. **Установите расширение** (см. выше)
2. **Откройте нужную страницу** с игрой/сайтом
4. **Приложение имеет два режима мониторинга мест:
   - занять первое освободившееся место
   - занять место или стать в очередь когда отмеченный (цвет метки можно выбрать) игрок/игроки сядут стол.
